    [分享发现] [独立开发前线] Vol.13 为什么说做网站依然是非常好的一个商业模式
    即使到了 2024 年，做网站依然是非常好的一个商业模式。网站可以为你带来持续的被动收入，相比 APP 或其他产品，网站有更多的优势，这也是 web 端长盛不衰的原因之一。 而我依旧非常看好网站，因为在长尾理论下，没有任何一家巨头能过垄断互联网内容，而当你的网站有大量的长尾内容时，可以吃到很多长尾流量，依靠长尾流量也可以存活的很好。 网站有以下独特的几点优势： 1 ）网站获取流量更容易 网站获取流量的方式主要有 3 种，1 ） seo ，通过搜索引擎获取流量，即用户在搜索引擎搜索某个关键词，然后你的内容出现在搜索结果中，从而给网站带来访问量。但这条路目前已经被百度给堵上了。 国内用户使用最多的搜索引擎还是百度，但百度现在基本不给独立站长流量，把绝大部分流量都倾斜给了百度系产品。所以 seo 很难做，目前只在长尾关键词上有一些机会，即这个搜索词非常长而且在百度系产品上没有很好的结果，而且热度不高，那么你才有可能排到前面。 2 ）直接访问，直接访问需要用户将你的网站加入收藏夹或者记住你的域名，直接在浏览器网址栏输入你的域名从而进行访问。直接访问的流量是最好的，说明你的网站对用户是有价值的，有吸引力的。 值得用户加入收藏夹或记住你的域名。这些对于用户来说都是有成本的。那么只有你的网站提供了足够多有价值的功能或者内容，用户才会愿意持续关注你。 3 ）社交媒体访问。当你把网站的内容分享到微信朋友圈或者今日头条，其他人通过你分享的链接来访问网站，这种就是社交媒体访问。社交媒体访问需要在社交媒体上分享大量的外链，但现在社交媒体对于内容的新鲜度有很大的考量，内容发布三五天之后一般就没有热度也没有自然流量了。 所以需要有持续的分享持续的推荐才能有一定的效果。而且需要多个社交媒体账号一起分享，效果才会比较好。如果有资源，可以在小红书和 B 站上找 UP 主付费推广，效果是不错的，至少可以让用户…  ( 2 min )
