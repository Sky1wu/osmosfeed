    [MacBook Pro] M1-M3 CPU 性能对比表格
    精力有限，简单做了个表格总结了一下发布至今各种丐版和满血版 M1 到 M3 CPU 部分性能，使用 CineBench R23 单核心和多核心分数作为参考。CPU 名字上的 - 代表丐版，往往屏蔽了 1-2 个核心，由于苹果的精心挑选，网上 kol 第一时间拿到的基本都是满血的 M3 和 M3 Max ，目前仅有 pcmag 一家有丐版的 M3 Pro 测试结果: https://www.pcmag.com/reviews/apple-macbook-pro-14-inch-2023-m3-pro ，所以里面 M3 Pro 满血和 M3 Max 丐版部分数据是推算。大部分数据来源于 https://www.check-mac.com/en/benchmark-cinebench_r23-10 选择 R23 是因为这基本上是 CPU 能达到的最完美理论极限，实际发挥来说，性能会比理论极限略差，尤其是多核心性能，几乎没几个 App 能根据核心数线性提升。 至于单核心性能，个人觉得这个比多核心性能甚至更为重要，很多开发和 App 其实就吃单核心，例如 Nodejs, Chrome 一个标签页等，只有单核性能不断提升，整体流畅度才会有质的变化，更能应对未来的系统升级和更复杂的 App ，而堆砌核心数边际提升递减，除了能在跑分和跑图里面有明显的优势，大多时候几乎都是闲置。 个人 2 cents：看了 pcmag 的测试，相信大家也能猜到苹果为什么只提供 M3 和 M3 Max 给 kol 测试，M3 Pro 多核心性能基本没太大的亮点，相较于 M1/M2 Pro 提升有限，尤其是考虑到用户如果买来丐版 M3 Pro 会发现多核心性能甚至不如满血版 M2 Pro 时，会有多么失望，所以也就刻意隐藏了。  ( 2 min )
